      50 ปุ่มลัดบนคีย์บอร์ดของ Mac ที่คุณควรรู้และใช้บ่อย

      โดย Zakura Kim 2 ปีที่แล้ว 36.1K ครั้ง

      สำหรับผู้ใช้อุปกรณ์ Mac ทั้งหมด โดยเฉพาะผู้ใช้ใหม่แนะนำเลยว่าควรเรียนรู้การใช้ปุ่มบนคีย์บอร์ด Mac ที่จะเป็นคีย์ลัดช่วยให้การทำงานของคุณนั้นรวดเร็วขึ้น เรามาชมกันดีกว่ากว่าคีย์ลัดบนคีย์บอด Mac ที่ควรรู้นั้นมีอะไรบ้าง

      50 ปุ่มลัดบนคีย์บอร์ดของ Mac ที่คุณควรรู้และใช้บ่อย

      ปุ่มลัดสำหรับระบบ

      1. Option + Command + ปุ่ม Power : เข้าสู่โหมดพักเครื่อง
      2. Control + Shift + ปุ่ม Power : เข้าสู่โหมดพักหน้าจอ (Sleep)
      3. Shift + Command + Q : ลงชื่อออก (Logout) จากบัญชี
      4. Control + Command + ปุ่ม Power : บังคับรีสตาร์ท โดยไม่บันทึกไฟล์ที่เปิดอยู่
      5. Control + Option + Command + ปุ่ม Power : ออกจากแอปทั้งหมดและปิดเครื่อง แต่จะมีหน้าต่างสอบถามการบันทึกไฟล์ที่เปิดอยู่และไม่ได้บันทึก
      6. Control + ปุ่ม Power : เรียกหน้าต่างเพื่อแสดงตัวเลือกรีสตาร์ท, พักเครื่อง หรือปิดเครื่อง

      ปุ่มลัดสำหรับการแสดงผลและ Dock

      1. Command + Space bar : แสดงและซ่อน Spotlight Search เพื่อเริ่มการค้นหาที่รวดเร็ว
      2. Option + Command + D : แสดงและซ่อน Dock ด้านล่าง
      3. Control + Command + Space bar : แสดงตัวแสดงอักขระ เช่น อิโมจิ สัญลักษณ์อื่นๆ
      4. Control + Command + F : แสดงแอปที่เปิดอยู่แบบเต็มหน้าจอ
      5. Command + Shift + 3 : ถ่ายภาพหน้าจอแบบเต็ม
      6. Command + Shift + 4 : ถ่ายภาพหน้าจอเฉพาะส่วนที่เลือก
      7. Command + Shift + 5 : เปิดเมนูควบคุมการถ่ายภาพหน้าจอ

      ปุ่มลัดสำหรับ Finder

      1. Shift + Command + C : เปิดหน้าต่างคอมพิวเตอร์
      2. Shift + Command + D : เปิดโฟลเดอร์ Desktop
      3. Shift + Command + F : เปิดโฟลเดอร์ล่าสุด
      4. Option + Command + L : เปิดโฟลเดอร์ดาวน์โหลด (Download)
      5. Shift + Command + O : เปิดโฟลเดอร์เอกสาร
      6. Shift + Command + U : เปิดโฟเดอร์ Utilities
      7. Shift + Command + R : เปิดหน้าต่าง AirDrop
      8. Shift + Command + G : เปิดหน้าต่างไปยังโฟลเดอร์
      9. Shift + Command + I : เปิดโฟลเดอร์ iCloud Drive
      10. Command + 1 :  ดูรายการในหน้าต่าง Finder เป็นไอคอน
      11. Command + 2 : ดูรายการในหน้าต่าง Finder เป็นรายการ
      12. Command + 3 : ดูรายการในหน้าต่าง Finder ในคอลัมน์
      13. Shift + Command + N : สร้างโฟลเดอร์ใหม่
      14. Option + Command + N : สร้างสมาร์ทโฟเดอร์ใหม่
      15. Option + Command + S : ซ่อนหรือแสดงแถบด้านข้างในหน้าต่าง Finder
      16. Option + Command + P : ซ่อนหรือแสดงแถบ Path ในหน้าต่าง Finder
      17. Command + เครื่องหมาย / : ซ่อนหรือแสดงแถบสถานะในหน้าต่าง Finder

      ปุ่มลัดสำหรับเอกสาร

      เมื่อใช้งานแอปพลิเคชันเกี่ยวกับเอกสาร สามารถใช้ปุ่มลัดได้ดังนี้

      1. Command + B : ปรับข้อความที่เลือกให้เป็นตัวหนา หรือเปิดปิดฟังก์ชันตัวหนา
      2. Command + I : ปรับข้อความที่เลือกให้เป็นตัวเอง หรือเปิดปิดฟังก์ชันตัวเอียง
      3. Command + U : ขีดเส้นใต้ข้อความที่เลือก หรือเปิปิดฟังก์ชันขีดเส้นใต้
      4. Fn + ลูกศรขึ้น : เลื่อนขึ้นไป 1 หน้า
      5. Fn + ลูกศรลง : เลื่อนลงไป 1 หน้า
      6. Fn + ลูกศรซ้าย : เลื่อนไปที่ส่วนเริ่มต้นเอกสาร
      7. Fn + ลูกศรขวา : เลื่อนไปที่ส่วนท้ายเอกสาร
      8. Command + S : บันทึกเอกสารปัจจุบัน
      9. Command + P : พิมพ์เอกสารปัจจุบัน
      10. Command + Z : ยกเลิกคำสั่งก่อนหน้า (Undo)
      11. Shift + Command + Z : กลับมาทำคำสั่งก่อนหน้าใหม่ (Redo)

      ปุ่ดลัดทัวไป

      1. Command + X : ตัดรายการที่เลือกพร้อมคัดลอกไปยังคลิปบอร์ด
      2. Command + C : คัดลอกรายการที่เลือก
      3. Command + V : วางรายการที่คัดลอกหรือตัด
      4. Command + A : เลือกรายการทั้งหมด
      5. Command + F : ค้นหารายการในหน้าต่างที่เปิดไว้
      6. Command + M : ย่อหน้าต่างด้านหน้าไปยัง Dock
      7. Command + W : ปิดหน้าต่างด้านหน้า
      8. Option + Command + Esc : บังคับปิดแอป
      9. Command + Tab : สลับแอปที่กำลังเปิดอยู่

      และนี่ก็เป็นคีย์ลัดหรือปุ่มลัดสำหรับ Mac ที่คาดว่าผู้ใช้ทั่วไปคงได้ใช้งานกันบ่อยและทำให้การทำงานบนอุปกรณ์ Mac นั้นทำได้ง่ายและรวดเร็วมากขึ้น ซึ่งยังมีคีย์ลัดอื่นๆ อีกมากมายให้เรียนรู้ สามารถดูคีย์ลัดแนะนำโดย Apple เพิ่มเติมได้ที่ ปุ่มลัดบนคีย์บอร์ดของ Mac
